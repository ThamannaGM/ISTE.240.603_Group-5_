# ISTE.240.603_Group-5_
Overview: EventHub is a smart web-based platform that helps users discover and manage events while
providing organizers with easy-to-use tools. Users can browse upcoming events, view details, and
register or cancel events. Organizers can create, update, and delete events, track registrations and
attendance, and access a mini dashboard with analytics. By combining event discovery,
management, and analytics, EventHub offers a complete end-to-end event management solution.
